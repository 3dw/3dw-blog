# 專案式學習的實踐

專案式學習（Project-Based Learning）讓學習圍繞一個真實問題或產出進行，很適合與自主學習結合。

## 什麼算「專案」？

一個小專案通常包含：

- **驅動問題**：例如「我們要怎麼讓班上更多人願意閱讀？」
- **產出**：海報、短片、一份報告、一個活動都可以。
- **過程**：查資料、討論、動手做、修改、分享。

::: info
專案不必很大，可以兩週完成一個「小專案」，重點是孩子有選擇與決策的空間。
:::

## 親師可以怎麼陪？

- **問問題**：少給答案，多問「你覺得呢？」「如果這樣做會怎樣？」
- **幫連資源**：書、網站、會的人，協助他們自己找到工具與資訊。
- **一起回顧**：專案結束後聊聊「學到什麼、下次想改哪裡」。
