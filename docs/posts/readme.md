# 關於本站

## 為什麼要留記錄

**Bestian:**
從一開始我們的會議每一場都可以記錄逐字稿，透過自動語音轉文字的方式把逐字稿記錄下來，讓後來參加的人可以接上前面的脈絡。

**Bestian:**
所以我們每一個會議都可以有逐字稿，逐字稿都可以編到網站，你覺得這有什麼好處？

**Water:**
好處是新來的人可以看到全部的脈絡。壞處是那麼多的逐字稿誰要看？

其實之後我們可以做一個總整工具，讓人在網站上按一個按鈕，AI 就把這些東西做摘要。以後瀏覽器也會自動做這件事，但在瀏覽器還沒做之前，我們可以手動提供這項服務。那是未來的事。

**Water:**
為何是未來的事？這不應該在我們建置網站時就考慮進去，只是一步步實作而已嗎？

**Bestian:**
當然。我已經考慮到了，但我意思是說實作需要時間，因為那是後端程式的事。

**Water:**
現在有 AI 幫忙，我相信速度會比以前全部靠手打 code 快很多。

**Bestian:**
