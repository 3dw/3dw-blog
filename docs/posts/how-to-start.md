# 如何開始自主學習

這篇給想帶孩子或學生嘗試自主學習的家長與老師，一個簡單的起步方式。

## 先從一小步開始

::: tip
不必一次推翻所有既有安排，可以從「一週一個主題由孩子選」或「一個小專案自己訂目標」開始。
:::

建議步驟：

1. **對話**：先跟孩子聊聊「你想學什麼／做什麼」，不批評、不立刻給答案。
2. **寫下來**：把目標、想用的方式、大概時間寫成簡單的計畫。
3. **一起回顧**：一週或兩週後，一起看「做到了什麼、卡在哪裡」，再調整。

## 常見的擔心

::: warning
「會不會都不學？」——通常來自我們對「學習」的想像太窄。先觀察：孩子在玩什麼、問什麼、查什麼，那往往就是他們正在學的。
:::

- 若孩子說「不知道要學什麼」，可以從興趣或生活問題出發（例如：為什麼會下雨？怎麼做一個遊戲？）。
- 若擔心進度，可以約定「最少要試的時數或產出」，其餘交給他們決定。
