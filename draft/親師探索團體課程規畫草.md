# 自主學習準備課程－教師篇(草案2026/03/23版)

深入瞭解自主學習的重要性，並為自己設定學習目標標。

## 課程基本資訊

- 人數：7~24人
- 時間：三小時一堂
- 期程：16堂課一期
- 課程進行方式：互動討論式、分組進行（大組、小組）

## 一、自我介紹（白紙操作）

- 畫一張你和學生的圖像
- 你認為教師這個角色的定位與功能為何？
- 為什麼你要當老師呢？
- 你對這個職業角色的感覺如何？曾經感到厭倦或困難嗎？
- 分享交流
- 作用：自我鏡照，並作為後續建立學習檔案之資料

## 二、建立討論文化（一）

建立課堂共識（實作）

## 三、建立討論文化（二）

了解討論文化的七項要素（講解）：

1. 聆聽
2. 尊重感受
3. 合作的態度（協同思考而非辯論）
4. 提意見要附理由
5. 用詞定義釐清
6. 界線清楚
7. 結論開放

親師生的界限概念與釐清操作

## 十、教師作為一個學習者（一）

### 學習的基本能力與配備

自主學習的四個階段：

- 解放 => 安全感
- 自我中心 => 界限
- 眼高手低 => 小階梯與成功經驗
- 自主成熟 => 知、行、悟的螺旋向上

正向的學習循環：

- 加分概念
- PDCA循環
  1. Plan => 計畫
  2. Do => 執行
  3. Check => 檢查
  4. Action（增、減、停）=> 行動（增加、減少、停止）
- 習慣的力量（覺察、評估、改變、自我酬賞）

## 十一、教師作為一個學習者（二）

- 學習動力初探（成長曲線圖）
- 麻痺經驗與重建
- 學習階梯的概念與實務（如何把大挑戰柝解成小步驟）
- 以自由數學的學習地圖為例

## 十二、教師作為一個學習者（三）

- 典範作用、教師角色與自我定位
- 個人書寫
- 小組分享與討論
